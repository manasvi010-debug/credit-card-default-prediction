## Project Overview
This project performs an exploratory data analysis (EDA) and builds a machine learning model to predict whether a credit card client will default on their payment next month. This is a critical task for security and risk management in the banking sector.

## Key Features
- Performed data cleaning and preprocessing on 30,000 client records.
- Analyzed correlations between repayment history, credit limits, and default probability.
- Visualized key trends using Matplotlib and Seaborn.
- Built a baseline classification model using Scikit-learn.

## Technologies Used
- **Python 3.9**
- **Pandas & NumPy** (Data manipulation)
- **Matplotlib & Seaborn** (Data visualization)
- **Scikit-learn** (Machine Learning)
- **Google Colab** (Development environment)
